# 安装

# npm安装

推荐使用 npm 的方式安装，它能更好地和 webpack 打包工具配合使用。

npm i ema-ui -S
1

# 引入Ema-UI

你可以引入整个 Ema-UI，或是根据需要仅引入部分组件。我们先介绍如何引入完整的 Ema-UI。

# 完整引入

在 main.js 中写入以下内容：

import emui from 'ema-ui'
import "ema-ui/dist/css/index.css"
Vue.use(emui)
1
2
3

# 按需引入

在 main.js 中写入以下内容：

import {Input,Radio} from 'ema-ui'
import "ema-ui/dist/css/index.css"
Vue.use(Input)
Vue.use(Radio)
